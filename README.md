🧠****AI Chatbot with Image Analysis (Streamlit + Gemini)****

This project is an AI-powered chatbot built using Streamlit and Google Generative AI (Gemini). It allows users to upload an image and ask questions about it. The chatbot analyzes the image and provides intelligent responses — specifically tailored for interior design suggestions.

**🚀 Features**

🖼️ Upload room images (JPG, JPEG, PNG)

💬 Ask questions based on the uploaded image

🎨 AI acts as an Interior Designer

⚡ Fast response using Gemini 2.5 Flash model

🌐 Simple and interactive UI using Streamlit

🛠️**Technologies Used**

     Python

     Streamlit

     Google Generative AI (Gemini API)

     PIL (Python Imaging Library)
